Qatja
=====

Java MQTT client library, conforms to MQTT 3.1.1

Based on the work by Andreas Göransson and David Cuartielles in the book "Professional Android Open Accessory programming with Arduino" and later [mqtt4processing](https://github.com/agoransson/mqtt4processing) and [mqtt4android](https://github.com/agoransson/mqtt4android).
